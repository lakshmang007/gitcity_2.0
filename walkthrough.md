# 🌍 GitCity: Virtual Earth Navigation

GitCity has been upgraded to a **Virtual Earth** experience. Each programming language is now its own **Country (Island)** in a vast voxel ocean, and you can zoom from the global map down into the specific repo buildings.

## 🚀 Navigation Controls

| Key | Action |
| :--- | :--- |
| **WASD** | Fly through the air (Globe or City view) |
| **Mouse** | Look around |
| **Scroll Wheel** | Change Altitude (Zoom in/out from the globe) |
| **Left Click** | **DIVE** — Instantly dive toward the point you are looking at |
| **M** | **MAP TOGGLE** — Switch between Global Map and Ground Level |
| **TAB** | **HUD TOGGLE** — Show/Hide repository statistics |
| **SPACE** | Jump (when on the ground) |
| **ESC** | Exit GitCity |

## 🏗️ Building Features (Ref: srizzon/git-city)
Inspired by the `git-city` project, our buildings now feature:
- 🪟 **Lit Windows**: Buildings show active windows (emissive voxels) based on development activity.
- 🗼 **Glowing Beacons**: Repositories with open issues have glowing glass beacons on their roofs.
- 🏷️ **Floating Labels**: 
    - **Global Scale**: High-altitude labels identify the "Land of [Language]".
    - **Local Scale**: Repository names fade into view as you approach specific buildings.

## 🏃 How to Run
1. Go to `c:/gitcity/build`
2. Run `gitcity.exe`
3. Enter your **GitHub Username** or paste your **Profile URL** (e.g., https://github.com/octocat)
4. Start exploring your code world from the sky!

---
> [!TIP]
> Use the **Left Click** while looking at an island from the sky to perform a cinematic "Dive" into that language's city.
